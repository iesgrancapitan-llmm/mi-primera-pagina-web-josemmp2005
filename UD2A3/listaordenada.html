Listas Ordenadas

HTML

HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es el componente más básico de la Web. Define el significado y la estructura del contenido web.

  1. Lanzamiento inicial: 1993
  2. Extendido de: SGML
  3. Extendido a: XHTML
  4. Tipo de formato: Lenguaje de marcado


XHTML

XHTML es, básicamente, HTML expresado como XML válido. Es más estricto a nivel técnico, pero esto permite que posteriormente sea más fácil al hacer cambios o buscar errores entre otros.

  1. Lanzamiento inicial: 26 de enero de 2000
  2. Última versión: Extensible HyperText Markup
  3. Tipo de formato: lenguaje de marcado
