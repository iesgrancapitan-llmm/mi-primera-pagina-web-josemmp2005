Editor WYSIWYG
Un editor WYSIWYG es el acrónimo de What You See Is What You Get en los que de manera visual se pueden colocar distintos elementos sobre una vista previa de la página, encargándose el programa de generar el documento web en lenguaje HTML. La aplicación es la encargada posteriormente de traducir lo que el desarrollador escribe y programar internamente la página con el código HTML. Ejemplo de editor WYSIWYG tenemos FrontPage y Dreamweaver.
